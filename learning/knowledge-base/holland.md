## 📚 霍兰德职业兴趣理论

### 什么是霍兰德职业测试？

**霍兰德职业兴趣量表**（Holland Codes / RIASEC）由美国心理学家约翰·霍兰德于 1959 年提出，是最具影响力的职业指导理论之一。

**核心假设**：
- ✅ 人可以分为 6 种职业兴趣类型
- ✅ 职业环境也可以分为 6 种类型
- ✅ 人职匹配度越高，满意度越高

**应用范围**：
- 高考志愿填报
- 职业规划设计
- 企业人才测评
- 职业咨询指导

---

## 🧩 6 种职业兴趣类型

### R - Realistic（现实型/实操型）

**核心特征**：喜欢动手操作，使用工具、机器

| 特征 | 描述 |
|------|------|
| 兴趣偏好 | 机械、建筑、运动、户外活动 |
| 能力特长 | 动手能力强、空间感知好 |
| 价值观 | 实用主义、看得见摸得着 |
| 典型职业 | 工程师、技师、运动员、农民 |
| 工作环境 | 户外、车间、实操场所 |

**测试题目示例**：
```
1. 我喜欢修理家电、家具等物品
2. 我擅长使用各种工具
3. 我喜欢户外活动胜过室内办公
```

---

### I - Investigative（研究型/探索型）

**核心特征**：喜欢思考、分析、探索未知

| 特征 | 描述 |
|------|------|
| 兴趣偏好 | 科学研究、理论分析、解决问题 |
| 能力特长 | 逻辑思维、分析能力、抽象思考 |
| 价值观 | 追求真理、知识、理解 |
| 典型职业 | 科学家、研究员、程序员、医生 |
| 工作环境 | 实验室、研究室、办公室 |

**测试题目示例**：
```
1. 我喜欢阅读科学类书籍
2. 我擅长解决复杂的逻辑问题
3. 我对未知事物充满好奇
```

---

### A - Artistic（艺术型/创造型）

**核心特征**：喜欢创造、表达、审美

| 特征 | 描述 |
|------|------|
| 兴趣偏好 | 文学、音乐、美术、表演 |
| 能力特长 | 创造力、想象力、审美能力 |
| 价值观 | 美、自由、自我表达 |
| 典型职业 | 作家、设计师、演员、音乐家 |
| 工作环境 | 工作室、舞台、自由空间 |

**测试题目示例**：
```
1. 我喜欢创作诗歌、故事或音乐
2. 我对色彩、构图有敏锐感知
3. 我讨厌按部就班的重复工作
```

---

### S - Social（社会型/助人型）

**核心特征**：喜欢帮助他人、与人交往

| 特征 | 描述 |
|------|------|
| 兴趣偏好 | 教育、咨询、服务、照顾 |
| 能力特长 | 沟通、倾听、理解他人 |
| 价值观 | 助人、合作、社会和谐 |
| 典型职业 | 教师、心理咨询师、护士、社工 |
| 工作环境 | 学校、医院、社区、服务机构 |

**测试题目示例**：
```
1. 我喜欢帮助他人解决问题
2. 我善于倾听和理解他人
3. 我觉得助人工作很有意义
```

---

### E - Enterprising（企业型/领导型）

**核心特征**：喜欢领导、影响、说服他人

| 特征 | 描述 |
|------|------|
| 兴趣偏好 | 管理、销售、政治、创业 |
| 能力特长 | 领导力、说服力、组织能力 |
| 价值观 | 成就、地位、影响力 |
| 典型职业 | 企业家、销售经理、政治家、律师 |
| 工作环境 | 办公室、商场、会议场所 |

**测试题目示例**：
```
1. 我喜欢担任团队领导
2. 我擅长说服他人接受我的观点
3. 我对商业活动很感兴趣
```

---

### C - Conventional（常规型/事务型）

**核心特征**：喜欢有序、规范、数据处理

| 特征 | 描述 |
|------|------|
| 兴趣偏好 | 文书、数据、财务、档案 |
| 能力特长 | 细致、准确、遵循规则 |
| 价值观 | 秩序、稳定、可预测 |
| 典型职业 | 会计、行政、图书管理员、数据分析师 |
| 工作环境 | 办公室、档案室、固定场所 |

**测试题目示例**：
```
1. 我喜欢按计划和流程做事
2. 我擅长处理数据和文档
3. 我注重细节和准确性
```

## 🎯 结果解读

### 六边形雷达图

```
        R
       / \
      /   \
     C-----I
     |     |
     E-----A
      \   /
       \ /
        S
```

**得分最高的 3 个类型** = 你的职业代码（如：ISA、REC 等）

### 常见职业代码

| 代码 | 类型组合 | 适合职业 |
|------|---------|---------|
| ISA | 研究 + 社会 + 艺术 | 大学教授、研究员、心理咨询师 |
| REC | 现实 + 企业 + 常规 | 工程师、项目经理、技术主管 |
| ASE | 艺术 + 社会 + 企业 | 广告创意、公关、媒体策划 |
| ESC | 企业 + 社会 + 常规 | 销售经理、人力资源、行政主管 |
| IRE | 研究 + 现实 + 企业 | 技术顾问、研发主管、专利律师 |
