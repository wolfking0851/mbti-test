## 📝 题目编写原则

### 语言风格

**✅ 正确**：
- 用生活化场景
- 每道题<30 字
- 避免专业术语
- 选项清晰明确

**❌ 错误**：
- 学术化表述
- 题目过长（>50 字）
- 双重否定句
- 模糊选项

---

### 题目示例对比

**好题目**：
```
1. 在社交聚会中，你通常：
A. 主动与很多人交流
B. 只与熟悉的人交流

✅ 优点：场景清晰，选项明确
```

**差题目**：
```
1. 当个体处于群体社交情境时，其行为倾向更符合：
A. 外向性特质主导的主动社交行为模式
B. 内向性特质主导的选择性社交策略

❌ 问题：学术化、太长、选项复杂
```

## 🎯 各维度题目模板

### E/I 维度题目

```
【场景类】
1. 周末你更喜欢：
A. 和朋友出去玩
B. 在家休息

【行为类】
2. 认识新朋友时，你：
A. 主动介绍自己
B. 等别人来介绍

【感受类】
3. 参加聚会后，你感觉：
A. 更有活力了
B. 有点累，需要休息
```

### S/N 维度题目

```
【信息处理】
1. 学习新知识时，你更喜欢：
A. 具体的例子和实际应用
B. 理论框架和概念模型

【注意力】
2. 你更擅长：
A. 记住具体细节
B. 理解抽象概念
```

### T/F 维度题目

```
【决策方式】
1. 做决定时，你更看重：
A. 逻辑和客观分析
B. 自己和他人的感受

【沟通风格】
2. 当朋友犯错时，你倾向于：
A. 直接指出问题
B. 先考虑对方的感受
```

## ⚠️ 避坑指南

### 常见错误

**错误 1：引导性题目**
```
❌ "你不喜欢社交场合，对吗？"
✅ "在社交场合中，你通常感到："
```

**错误 2：双重问题**
```
❌ "你喜欢阅读和运动吗？"
✅ "你喜欢阅读吗？"
✅ "你喜欢运动吗？"
```

**错误 3：社会期许偏差**
```
❌ "你会帮助有需要的人吗？"（大部分人会选"会"）
✅ "当朋友遇到困难时，你通常："
```

**错误 4：绝对化表述**
```
❌ "你总是..."
❌ "你从不..."
✅ "你通常..."
✅ "你倾向于..."
```
