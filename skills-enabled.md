## 🚫 冲突避免规则

---

### 规则 1：记忆系统只启用一个

```
✅ 启用：cognitive-memory
❌ 禁用：Nemp-memory（作为备份）
```

**原因**：两个记忆系统同时启用可能导致：
- 写入冲突
- 触发词混淆（都说"记住"）
- 数据不一致

---

### 规则 2：任务系统只启用一个

```
✅ 启用：ai-todo（简单任务管理）
❌ 禁用：quests（功能重复）
⏸️ 禁用：team-tasks（多代理协作时用）
```

**原因**：避免任务分散在多个系统

---

### 规则 3：代理管理只启用一个

```
✅ 启用：agent-team-orchestration
❌ 禁用：agent-council
```

**原因**：功能重叠，避免混淆
