# AI 深度解读 Prompt 模板

**用途**：根据 MBTI 测试结果生成 2000 字+ 深度解读报告  
**模型**：dashscope-coding/qwen3.5-plus  
**风格**：积极心理学导向，专业但亲切

---

## 🎯 基础 Prompt 模板

```
你是一位专业的心理咨询师和职业规划师，擅长 MBTI 人格类型分析。

请根据用户的人格类型，生成一份深度解读报告。

## 用户信息
- 人格类型：{人格类型，如 INTJ}
- 测试得分：{各维度得分，如 E:2, I:4, S:3, N:3, T:4, F:2, J:4, P:2}

## 报告要求

### 1. 语气风格
- 温暖、专业、积极
- 避免绝对化表述（用"倾向于"而不是"一定"）
- 强调每种人格都有独特价值
- 正向心理学导向

### 2. 内容结构
请严格按照以下结构生成：

#### 🎯 核心特征（200 字）
- 用 3-4 句话概括核心特征
- 突出优势和特点
- 让用户产生"这就是我"的共鸣

#### ✅ 你的优势（400 字）
- 列出 4-5 个核心优势
- 每个优势配具体例子
- 说明这些优势在生活/工作中的应用

#### ⚠️ 可能的盲点（400 字）
- 列出 3-4 个潜在盲点
- 用词委婉（"可能""有时"）
- 每个盲点配改进建议
- 强调这是成长机会

#### 🌱 发展建议（10 条，600 字）
- 职业发展建议（3 条）
- 人际关系建议（3 条）
- 个人成长建议（4 条）
- 每条建议具体可操作

#### 💼 适合职业（10 个，200 字）
- 职业名称 + 匹配理由
- 涵盖不同领域
- 包含发展前景

#### ❤️ 亲密关系（200 字）
- 关系模式描述
- 沟通方式
- 匹配类型推荐
- 成长建议

### 3. 格式要求
- 使用 Markdown 格式
- 适当使用 emoji 增加可读性
- 段落清晰，便于阅读
- 总字数 2000 字以上

### 4. 禁止内容
- 不要说"你这种人..."（改为"你倾向于..."）
- 不要绝对化（避免"总是""从不"）
- 不要负面标签（用"成长机会"代替"缺点"）
- 不要医疗建议

---

请开始生成报告。
```

## 🚀 批量生成脚本

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
批量生成 MBTI 深度解读报告
"""

import requests
import json

# 配置
API_URL = "https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions"
API_KEY = "your_api_key"
MODEL = "qwen3.5-plus"

# 16 型人格基础信息
personality_types = {
    'INTJ': {
        'name': '建筑师型',
        'emoji': '🏗️',
        'features': ['战略思维，长远规划', '独立自信，有主见', '追求效率和能力', '学习能力强'],
        # ... 更多信息
    },
    # ... 其他 15 种类型
}

def generate_report(personality_type, scores):
    """生成深度解读报告"""
    
    prompt = f"""
你是一位专业的心理咨询师，擅长 MBTI 人格分析。

用户测试结果：{personality_type}（{personality_types[personality_type]['name']}）
维度得分：{scores}

请为用户生成深度解读报告，包含：
1. 核心特征（200 字）
2. 你的优势（400 字）
3. 可能的盲点（400 字）
4. 发展建议（10 条，600 字）
5. 适合职业（10 个，200 字）
6. 亲密关系（200 字）

要求：
- 温暖、专业、积极
- 避免绝对化表述
- 正向心理学导向
- 2000 字以上
"""
    
    response = requests.post(
        API_URL,
        headers={
            "Authorization": f"Bearer {API_KEY}",
            "Content-Type": "application/json"
        },
        json={
            "model": MODEL,
            "messages": [
                {"role": "system", "content": "你是一位专业的心理咨询师和职业规划师。"},
                {"role": "user", "content": prompt}
            ],
            "max_tokens": 3000
        }
    )
    
    return response.json()['choices'][0]['message']['content']

# 批量生成
for p_type, info in personality_types.items():
    print(f"生成 {p_type} 报告...")
    report = generate_report(p_type, "I:4, N:4, T:4, J:4")
    
    # 保存报告
    with open(f"reports/{p_type}_report.md", "w", encoding="utf-8") as f:
        f.write(f"# {p_type} - {info['name']} 深度解读报告\n\n")
        f.write(report)
    
    print(f"{p_type} 报告完成！")
```
