## 📋 新的规划模板（所有项目必须遵守）

### 1. 参与人员规划
- 每个阶段至少 5-6 个娃参与
- 明确角色：研究员、工程师、审查员、测试员、汇报者、监督者
- 不能单打独斗

### 2. 协作流程（标准化）
```
研究 → 评审 → 实施 → 审查 → 修改 → 测试 → 汇报
```

### 3. 复核机制
- 设计方案：四娃 + 大娃复核
- 代码：四娃复核
- 测试报告：大娃复核

### 4. 修改机制
- 设计问题 → 研究员修改
- 代码问题 → 工程师修改
- 审查不通过 → 打回重做

### 5. 监督机制
- 大娃每 5 分钟检查进度和健康
- 发现问题自动告警（带解决方案）
- 解决不了找老板
