```

```

**优化后**（具体）：

**优化点**：
1. 增加具体场景（咖啡厅）
2. 增加人物动作（指出问题、倾听）
3. 增加具体物品（笔记本、咖啡）
4. 增加光影描述（柔和自然光）
