## 🔧 配置说明

### 广告配置
编辑 `js/main.js`:
```javascript
const adConfig = {
    enabled: true,      // 上线时改为 true
    adUnitId: 'xxx',    // 填入广告位 ID
};
```

### 服务器端口
默认 8888，修改：
```bash
python3 -m http.server 9000  # 改为 9000 端口
```
